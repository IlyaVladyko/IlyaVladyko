## 🚀 Main Project —  [NMC](https://github.com/IlyaVladyko/NMC)

An extended Monte Carlo framework that goes beyond classical radiative transfer.
| Feature | What it means |
|---|---|
| Raman photon generation | Nonlinear scattering modeled from first principles |
| Spectral modeling | Proper handling of wavelength-dependent effects |
| Subsurface path tracing | Complex 3D geometry support |
| Metal + Unified Memory | GPU-accelerated on Apple Silicon |
| Smarter sampling | Variance reduction to cut runtime |

In simple terms:  
I try to make photon transport simulations more realistic, more scalable, and more useful for real-world sensing.

**Applications:** biomedical optics · subsurface sensing · photodynamic therapy · precision agriculture · food quality
